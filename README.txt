~~~READ ME~~~
Instructions for running code.

1.	Extract all .py files from the submitted zip file into one folder.
2. Open the main.py file in IDLE and running

HOW TO PLAY
1. After running the file, enter the difficulty of the game you would like(enter the number corresponding to the difficulty)
2. Co-ordinate entry when prompted:
	x1: -> horizontal co-ordinate of piece you want to move
	y1: -> vertical co-ordinate of piece you want to move
	x2: -> horizontal co-ordinate of destination square
	y2: -> vertical co-ordinate of destination square
